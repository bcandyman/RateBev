# RateBev

## Description
RateBev is a web application used to allow user ratings of beverages to be captured. These ratings may be revisted by the user at any time in the future.

## Technologies
- Node.js
    - Sequelize
    - Express
    - JwtWebToken
    - BCrypt
- MySql
